Notas bootstrap:
Para usar el responsive y los tooggles se de be usar la importacion
import "bootstrap/dist/css/bootstrap.min.css"
import 'bootstrap/dist/js/bootstrap.bundle.min';
para el error de popper leer : https://stackoverflow.com/questions/57459917/how-to-fix-this-error-module-not-found-cant-resolve-popper-js
y para solventarlo instalar pooper : npm install popper.js --save

para alinear los elementos a la derecha se usa en el ul : ms-auto

leer referencia para mas usos: 
https://stackoverflow.com/questions/19733447/bootstrap-navbar-with-left-center-or-right-aligned-items
